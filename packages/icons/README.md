# @absolo/icons

Icon set for Absolo frontends.

```tsx
import { Boxes, Database, Sun } from '@absolo/icons';
import { AbsoloWordmark } from '@absolo/icons/brand';
```

- Re-exports a curated subset of [Lucide](https://lucide.dev) icons (kept narrow for bundle hygiene).
- Provides Absolo brand marks in `brand.tsx`.

Plan reference: [`../../docs/plans/04-design-system-d1e00e.md`](../../docs/plans/04-design-system-d1e00e.md).
